belajar git git init untuk mengecek file yang ada di folder

setelah di git init kita bisa cek status file menggunakan git status

untuk menambahkan file dan mengubah file kita menggunakan git add .

setelah di ubah kita bisa menyimpan dengan menggunakan perintah git commit -m "deklarasi"

lalu menambahkan ke github dengan membukan new repository lalu ketik perintah git remote add origin URLREPSITORY

setelah itu ketik perintah git push -u origin master
